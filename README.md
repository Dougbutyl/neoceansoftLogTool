## Dependency
Add it in your root build.gradle at the end of repositories:
``` Java
 	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
 ```
 and then add dependency
``` Java
 	dependencies {
	        implementation 'com.github.Dougbutyl:neoceansoftLogTool:1.0'
	}

 ```
